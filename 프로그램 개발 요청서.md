## thread sns 자동화 마케팅 프로그램 개발 요청서
목적: '호지프릭스' 티백 제품 판매 촉진을 위한 스레드 sns 자동화 
목표 : 같은 성격, 같은 말투를 유지하면서도 겹치지 않게 일상적인 글과 제품을 홍보하는 글, 나를 태그해서 내 제품을 올려준 사람에 대한 답글 등을 자동으로 달아주면서 내가 확인하고 ok만 하면 자동으로 글이 올라가는 식으로 자동화 thread sns 마케팅 프로그램 개발
동작방법: 
1. 스레드 계정 데이터 수집, 나를 태그한 글이나 답글은 자동으로 수집
2. 브랜드 말투/성격 기준으로 글 자동 생성답글의 경우 상활별 답글 초안 생성
3.  중복·과한 홍보·금지 표현 검사
4. 초안.md에 초안 및 타임로그 등 저장한 후 Slack으로 전송
5. Slack에서 내가 확인 후 `OK` 체크
6. 체크된 글 투고목록.md 또는 답글목록.md 저장 후 Threads API로 자동 게시
